# Demo Video Guide

This document provides guidance on creating a demo video for Clarity.

## Video Specifications

- **Duration**: 30-60 seconds
- **Format**: MP4 (H.264)
- **Resolution**: 1920x1080 (Full HD) or higher
- **Frame Rate**: 30 FPS
- **Aspect Ratio**: 16:9

## Content Outline

### Opening (0-5 seconds)
- Quick logo/branding animation
- Tagline: "Clarity - AI-Powered Productivity Tracker"

### Core Features (5-45 seconds)
1. **Screen Recording** (5-10s)
   - Show the recording button
   - Demonstrate automatic screenshot capture
   - Highlight minimal CPU usage

2. **Activity Timeline** (10-20s)
   - Navigate to Trace page
   - Show timeline with AI summaries
   - Highlight activity tags and metrics

3. **Statistics Dashboard** (20-30s)
   - Show statistics page
   - Highlight charts and visualizations
   - Show API usage metrics

4. **Daily Summary** (30-40s)
   - Show Summary page
   - Demonstrate AI-generated insights
   - Show comparison charts

5. **Privacy Emphasis** (40-45s)
   - Highlight "All data stays local"
   - Show settings page
   - Emphasize privacy-first approach

### Closing (45-60 seconds)
- Call to action: "Try Clarity today"
- GitHub link and star button
- Final branding

## Recording Tips

1. **Use Screen Recording Software**:
   - macOS: QuickTime Player or ScreenFlow
   - Windows: OBS Studio or Camtasia
   - Linux: OBS Studio or SimpleScreenRecorder

2. **Preparation**:
   - Clean desktop background
   - Use realistic demo data
   - Ensure smooth transitions
   - Hide sensitive information

3. **Editing**:
   - Add smooth transitions
   - Include text overlays for key features
   - Add background music (optional, keep it subtle)
   - Add captions for accessibility

4. **Optimization**:
   - Compress for web (keep file size reasonable)
   - Create multiple versions (full, short, GIF)
   - Optimize for different platforms

## Distribution

- **GitHub**: Upload to repository releases or use GitHub Pages
- **Product Hunt**: Upload as gallery video
- **YouTube**: Create a channel for product demos
- **Twitter/X**: Create shorter clips (15-30s) for social media

## Example Script

```
[0-5s] Logo animation + "Clarity - Track your digital activity with AI insights"

[5-10s] "Start recording with one click" - Show recording button

[10-20s] "View your activity timeline" - Navigate Trace page, show summaries

[20-30s] "Get detailed statistics" - Show charts and metrics

[30-40s] "AI-powered daily summaries" - Show Summary page with insights

[40-45s] "All data stays local - Your privacy matters" - Show settings

[45-60s] "Try Clarity today - Open source and free" - GitHub link
```

## Tools

- **Recording**: OBS Studio (free), ScreenFlow (macOS), Camtasia
- **Editing**: DaVinci Resolve (free), Final Cut Pro, Adobe Premiere
- **Compression**: HandBrake (free), FFmpeg
